# python-fundamentals

Runnable Jupyter companion to **[python-from-zero](https://python-from-zero.pages.dev)** and the
*Python for VLSI* book. Each notebook mirrors one chapter of the book: concept, short VLSI
example, three exercises with solutions, and a recap.

- Python 3.9+, **stdlib only** — no external dependencies.
- Self-contained: any file an example needs is written inline before being read.
- All cells executed with embedded outputs, so the notebooks render readably on GitHub.

## Numbering convention

Filenames use the **book chapter number** (1–17). The book covers 17 chapters; only the 14
chapters with hands-on Python content have a notebook here, so numbers 5, 7, and 9 are skipped
on purpose. The website lesson order differs slightly — see the table below for the mapping.

## Running locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install jupyter
jupyter lab
```

## Rebuilding the notebooks

The notebooks are generated from small Python specs in `scripts/`:

```bash
cd scripts
python3 build_all.py
```

To re-execute every notebook and refresh embedded outputs:

```bash
for f in Chapter_*.ipynb; do
  jupyter nbconvert --to notebook --execute --inplace "$f"
done
```
