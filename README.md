# python-fundamentals

Runnable Jupyter companion to **[python-from-zero](https://python-from-zero.pages.dev)** and the
*Python for VLSI* book. Each notebook mirrors one book chapter: concept, short VLSI example,
three exercises, recap.

- Python 3.9+, **stdlib only** — no external dependencies to install.
- Every notebook is **self-contained**: files used in examples are written inline before being read.
- All cells have embedded outputs, so the content is readable on GitHub without running anything.

## Running locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install jupyter
jupyter lab
```

Then open any `Chapter_*.ipynb` file.

## Rebuilding the notebooks

The notebooks are generated from small Python specs in `scripts/`:

```bash
cd scripts
python3 build_all.py
```

To re-execute every notebook and refresh embedded outputs:

```bash
for f in Chapter_*.ipynb; do
  jupyter nbconvert --to notebook --execute --inplace "$f"
done
```
